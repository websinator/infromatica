Dagelijkse Meme's

Onze opdracht

Wij moesten voor het vak informatica een website bouwen. Deze website moest aan een paar eisen voldoen zodat wij leren te programeren met HTML, CSS en javascript. Hiervoor kregen wij het programma notepad++, maar wij hebben ook het programma Brackets gebruikt omdat dit een makkelijkere manier was van coderen. Dit komt omdat dan soms stukjes van de code worden aangegeven waardoor je sneller de code kon maken.

Wat kan je vinden op deze pagina

Op onze meme pagina kan je meme’s vinden. Hier worden dagelijks meme’s geplaatst. Je kan sorteren op de meme’s die vandaag, gisteren of al langer geleden zijn gepost. Er is ook een mogelijkheid om contact met ons op te nemen. Je kunt al deze tabladen openen via ons menu, waardoor je direct naar de pagina waar je wilde zijn wordt gestuurd.
